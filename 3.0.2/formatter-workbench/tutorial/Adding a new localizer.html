In the previous activity, we referenced a custom localizer supplied with this app. Now we’re going to create a simple localizer from scratch.

Adding a new localizer

For a grid cell to use a localizer to format its value, three coding steps are required:
  1. Define the localizer
    Code the localizer in the Localizers tab's editor.
  2. Register the localizer with the grid
    Use the Localizers tab’s grid.localization.add(…) button.
  3. Bind the localizer to some cells
    Cells are loosely bound to the localizer, by name, through their format property.* To set the property declaratively, use the State tab's editor.

In the activity on the next page we’ll perform these three steps.
*For historical reasons, the property name is format rather than localizer.
