Cell editors + localizers

When an editable cell is bound to a localizer, the cell editor will use it to format and deformat the value:
  1. On open, the cell editor invokes localizer.format to format the value
  2. On save, the cell editor invokes localizer.parse on the edited value to convert it back to its native type
