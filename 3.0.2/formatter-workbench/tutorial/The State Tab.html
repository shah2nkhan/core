State tab

This tab provides an editor for grid state.
  • The state is stored as a plain object
  • This object is used to set grid properties declaratively
  • Hypergrid has API calls for loading and saving state
  • Grid properties are generally own members of the grid.properties object (that is, not including its prototype chain, which contains defaults, theming, etc.)
To make editing easier, this editor, like the Data tabâ€™s editor, uses eval to parse straight JavaScript, not JSON. You do not need to concern yourself with strict adherence to the JSON standard (double quotes, primitive types, _etc._). And you can also comment out lines, which is not allowed in JSON.
To set grid properties with the grid.setState(state) method is simply a matter of adding members to the root of your state object: 
state = {
    propName: propValue
}
