Bind a cell to a localizer

6.1. Bind cells of prevclose column to ft-in localizer:

In its default state, none of the cells in the grid at left are formatted. In this activity, we reset the grid to its default state (if needed) and then add formatting to one of the columns using a supplied custom localizer.

  1. If needed: Switch to the Data tab and click its button to reload the default data
  2. Switch to the State tab
  3. If needed: Click the tab’s button to reload the default state
  4. Add the format grid property to the prevclose column: 
      prevclose: {
      format: "ft-in",
      halign: "right"
  }
  5. Click the grid.setState(…) button
  6. The values in this column are now formatted as feet and inches
The format property binds the cell to a localizer. The value must be the name of a registered localizer, in this case ft-in, a custom localizer packaged with this workbench app. The value displayed in the cell is formatted by localizer.format.
