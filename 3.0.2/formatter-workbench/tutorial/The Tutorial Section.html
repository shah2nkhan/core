Workbench interface: Tutorial section

The tutorial content area has multiple pages. Every page has a Tutorial tab, with some pages also having a Code Snippets tab.

Tutorial tab

This is the tab youâ€™re reading from right now.

The tutorial is split into easily digestible, page-size chunks. Depending on your window size, you may need to scroll down to see the whole page.

To move from page to page:
  • Click the ◀ or ► buttons above to go to the previous or next page, respectively
  • Press the ← or → keys on your keyboard which also go to the previous page and next page, respectively*
  • Drag the slider found between the page buttons for random access to any page

Code Snippets tab

This tab contains code snippets for cutting-and-pasting and only appears when the current tutorial page has such content.
Click the ► button (above) to go to the next page.
* Note however that these arrow keys cannot be used for paging when a text field has the focus because in that case they are used for editing.
