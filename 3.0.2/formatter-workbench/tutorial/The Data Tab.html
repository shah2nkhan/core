Data tab

This tab provides an editor for the grid’s data source.
  • The data is structured as an array of dataRow objects.
  • The grid data reflects the initial contents of this tab.
  • After editing the data in this tab, bind it to the grid by clicking grid.setData(…).
  • This Workbench app implements two-way binding: That is, the grid is bound back to the Data tab so any edits made in the grid itself are automatically reflected in the Data tab.
In actual practice, the data object passed to setData has typically been parsed from a character string in JSON format (by JSON.parse).

To make editing easier, this editor uses eval to parse straight JavaScript, not JSON. You do not need to concern yourself with strict adherence to the JSON standard (double quotes, primitive types, _etc._). And you can also comment out lines, which is not allowed in JSON.
